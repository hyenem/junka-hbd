# 🦫 쭌카 생일파티 (junka-hbd)

준영(애칭 **쭌카**)의 28번째 생일(980619) 축하 멀티페이지 사이트.
모바일 우선 · 의존성 0 vanilla HTML/CSS/JS · GitHub Pages 호스팅.

**입장은 너드 관문(대학수학 문제)을 풀어야 가능** — 못 풀면 모든 페이지가
관문(`gate.html`)으로 자동 리다이렉트됩니다.

페이지마다 컨셉이 다름:

| 페이지 | 내용 | 컨셉 |
|---|---|---|
| `gate.html` | 너드 관문 (입구) — 대학수학 무한급수 | 칠판 교실 |
| `index.html` | 홈 / 생일 초대장 + 놀거리 허브 | 따뜻한 쿼카 파티 |
| `exchange.html` | 준영증권거래소 + IPO 청약/주주증서 | 트레이딩 터미널 (유일) |
| `augment.html` | 롤토체스 증강체 뽑기 | 헥스테크 매직 |
| `gg.html` | 쭌카.GG 전적 | e스포츠 대시보드 |
| `climb.html` | 클라이밍 등반 게임 | 볼더링장 |
| `bingo.html` | 파티 빙고 | 레트로 팝 |
| `roulette.html` | 파티 룰렛 | 카지노/카니발 |

`confetti.js` 는 모든 페이지가 공유하는 색종이 모듈.
진행 상태는 `localStorage`(`junka:*`)에 저장.
