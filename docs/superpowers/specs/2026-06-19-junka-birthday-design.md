# 쭌카 28th 생일 축하 페이지 — 설계 문서

작성일: 2026-06-19

## 목적
준영(애칭 **쭌카**)의 28번째 생일(980619)을 축하하는 웹페이지. 생일 파티
초대장 역할 + 파티에 모인 사람들이 각자 폰으로 즐길 도박성 미니게임 제공.

## 대상 인물 (콘텐츠 소재)
- 경기과학고 → 서울대 수학과 졸업한 영재
- 삼성전자 개발자
- 980619생, 쿼카(quokka)를 닮음 → 애칭 "쭌카"
- 이성적이지만 착함 / 너드처럼 보이지만 사실 사회성이 강함
- 취미: 롤(LoL), 롤토체스(TFT, 전국구 실력), 클라이밍

## 핵심 결정사항
- **플랫폼**: 정적 사이트, **GitHub Pages 배포**
- **디바이스**: 모바일 우선 (각자 폰으로 접속)
- **상태 공유**: 서버 없음 → **각 폰에서 독립 실행** (실시간 공유 리더보드는 v2)
- **톤**: 쿼카 귀여움 × 너드/밈 감성 믹스
- **언어**: 한국어
- **기술 스택**: 단일 정적 사이트. 의존성 없는 vanilla HTML/CSS/JS
  (빌드 도구 없이 GitHub Pages에 바로 올라가도록). 파일 3개 구성:
  `index.html` / `styles.css` / `app.js`.

## 디자인 컨셉 & 미감
- **통일 컨셉**: 페이지 전체를 **"쭌카 증권거래소(JUNKA EXCHANGE)"** 로
  통일. 따뜻한 쿼카 감성으로 데운 트레이딩 터미널. 4개 섹션이 하나의
  거래소 내러티브로 묶임:
  - Hero = "오늘 쭌카 28세 상장(IPO)" 초대장
  - 거래소 = 메인 트레이딩 플로어
  - 분석 리포트 = "종목 분석 리포트: JUNKA"
  - 룰렛 = "변동성 룰렛"
- **타이포그래피**: 디스플레이 `Black Han Sans`(임팩트 한글),
  본문 `IBM Plex Sans KR`, 숫자/티커 `IBM Plex Mono` (Google Fonts).
- **컬러**: 다크 허니·앰버 베이스 + 트레이딩 그린/레드 기능색.
  (보라 그라데이션 등 흔한 AI 톤 회피)
- **시그니처 요소**: 상단에 흐르는 **쭌카 종목 티커**, 숫자
  카운트업, 손그림 SVG 쿼카 마스코트.

## 페이지 구성 (10개 섹션 + 상단 티커, 한 페이지 스크롤)

### 0. 상단 티커 (고정)
- 쭌카 종목·팩트가 흐르는 마퀴 (예: "롤토체스코인 ▲▲ 전국구",
  "980619 상장"). 거래소 컨셉의 시그니처.

### 1. 🎉 초대장 (Hero / 상장)
- "쭌카" 대형 타이틀(Black Han Sans) + 28th Birthday + 980619
- 손그림 SVG 쿼카 마스코트 (파티 고깔 착용)
- 색종이(confetti) 애니메이션
- 케이크 촛불 탭하면 꺼지는 인터랙션 → 색종이 + 축하 토스트
- 파티 정보: 날짜(오늘 2026-06-19), 장소(석암생소금구이) — 시간 없음
- 초대 문구 한마디
- "거래소 입장 →" CTA

### 2. 📈 준영증권거래소 (메인 도박 콘텐츠) ⭐
- 시작 가상머니: `쭌카머니 ₩1,000,000`
- 종목 (준영 인생 테마, 변동성 캐릭터 부여):
  - `쿼카전자` — 삼성 패러디, 저변동 안정형
  - `롤토체스코인` — 초고변동, 가끔 떡상(전국구)
  - `서울대수학지수` — 중변동
  - `클라이밍홀딩스` — 중변동
  - `경기과학고밈` — 변동성 랜덤
- 매 라운드 가격 변동 + 랜덤 이벤트 토스트
  (예: "쭌카 롤토체스 1등! 롤토체스코인 +30% 🚀",
   "야근 발생, 쿼카전자 -12% 📉")
- 매수/매도 버튼 (모바일 큰 버튼)
- **총 10라운드(DAY 1/10)**, 종료 후 최종 평가금액 → 수익률 → 칭호
  (예: "쭌카 워렌버핏" / "쭌카 개미지옥")
- 결과 화면 캡처/공유 안내 (각자 폰 비교용)

### 3. 🔬 종목 분석 리포트 (캐릭터 카드)
- 증권 리서치 리포트 풍의 스탯 카드 한 장
- 헤더: 티커 `JUNKA`, ★★★★★, "강력 매수" 뱃지
- 스탯: 이성 99 / 착함 95 / 사회성 92 / 쿼카력 100 /
  너드위장력 88 (스크롤 진입 시 막대 채워짐)
- 태그: 경기과학고·서울대 수학과·삼성전자·클라이밍·롤토체스 전국구
- "애널리스트 의견" 너드밈 카피

### 4. ♟️ 롤토체스 증강체 뽑기
- TFT 증강체 UI 패러디: 3장 중 1택
- 카드 내용은 쭌카 생일 운세/덕담 (예: "무한 떡상 — 가는 곳마다 1등")
- 등급(Silver/Gold/Prismatic)별 색상, 가챠 손맛
- 하루 1회 제한(localStorage), 뽑은 결과 캡처용으로 표시

### 5. 🎮 쭌카.GG (전적 패러디)
- op.gg / 롤토체스 전적창 패러디한 가짜 프로필
- 티어/스탯 개그 (예: "솔랭 챌린저 · 인성 MMR 9999 · 야근 KDA")
- 최근 전적 리스트(승/패) 밈 카피

### 6. 🧗 클라이밍 등반 미니게임
- 탭(또는 버튼) 연타로 벽 올라가기, 제한시간/홀드 표현
- 정상 도착 시 축하 메시지 + 색종이
- 최고 높이 기록 localStorage 저장

### 7. 🧮 너드 관문 (수학 이스터에그)
- 서울대 수학과 헌정 깜찍 수식 퀴즈 (정답: 28 = 쭌카 나이)
- 정답 입력 시 히든 메시지/연출 해금
- 가벼운 이스터에그 (틀려도 힌트)

### 8. 🅱️ 쭌카 빙고 (파티용)
- 파티 중 일어날 일로 채운 5x5(또는 3x3) 빙고판
  (예: "롤토체스 얘기 시작", "쿼카 같다는 말 나옴", "코드 얘기함")
- 각자 폰으로 탭 체크, 빙고 완성 시 축하 연출
- 체크 상태 localStorage 저장

### 9. 📜 IPO 청약 / 주주증서
- 이름 + 축하 한마디 입력 → 랜덤 보유주식수 박힌 쭌카 주주증서 발급
- 증서는 캡처용 굿즈 형태 (날짜·발행번호·도장 느낌)
- 거래소 컨셉의 마무리. 로컬 저장(영속 공유는 v2)

### 10. 🎡 변동성 룰렛
- 돌림판(룰렛) 랜덤 추첨
- 술값 내기 / 벌칙 정하기 용도
- 항목 직접 추가/삭제 가능 (이름·벌칙 입력)
- 돌리면 애니메이션 후 당첨 항목 강조
